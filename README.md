## 🚀 What This Project Does

A **role-based multi-agent AI system** built from scratch — no LangChain, no paid APIs. A supervisor agent delegates tasks across specialized agents that communicate to complete work collaboratively.

---

## ✅ Features

- 🧠 **Supervisor pattern** — coordinates task delegation across agents
- 🔍 **Research Agent** — gathers information relevant to the task
- 📊 **Analysis Agent** — processes and interprets gathered data
- ✍️ **Writer Agent** — produces the final output
- 🦙 **Local inference** — runs entirely on Ollama + LLaMA 3.1, zero paid APIs

---

## 🚀 Run Locally

\`\`\`bash
git clone https://github.com/tahirAliicodes/AgenticAI-MultiAgentOrchestration.git
cd AgenticAI-MultiAgentOrchestration
python main.py
\`\`\`

Requires [Ollama](https://ollama.com) running locally with the \`llama3.1\` model pulled.
